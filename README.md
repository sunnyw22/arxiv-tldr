# Research Radar

Personalized paper discovery and screening for researchers. Monitors arXiv, ranks papers against your research profile, and delivers evidence-backed summaries.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Configuration

Copy `config/config.example.yaml` to `config/config.yaml` and edit to match your research interests.
